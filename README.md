# PulseNet - Social Media Web App

PulseNet is a full-featured social media web app built with HTML, CSS, and vanilla JavaScript.

## Features

- Authentication (local register/login simulation)
- Create posts with text, images, and hashtags
- Like, comment, repost, share, and bookmark posts
- Follow/unfollow users
- Explore/search users and posts
- Direct messages
- Story updates
- Notifications feed
- Profile editing
- Dark mode toggle

## Run locally

Because it uses browser features like `crypto.randomUUID()`, serve it over HTTP:

```bash
python3 -m http.server 4173
```

Then open: `http://localhost:4173`
